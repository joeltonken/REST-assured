# Testes Rest-Assured

Este repositório contém um projeto de exemplo que demonstra como usar o Rest-Assured para testar API do [ServeRest](https://serverest.dev/).

## Configuração do Ambiente

Certifique-se de ter as seguintes ferramentas instaladas em sua máquina:

- Java JDK 17
- Maven
- Git

## Como Executar o Projeto

1. Clone este repositório em sua máquina local:

```bash
https://github.com/joeltonken/REST-assured
```

2. Navegue até o diretório do projeto:

```bash
cd nome-do-repositorio
```

3. Compile o projeto usando o Maven:

```bash
mvn clean install
```

4. Execute os testes utilizando o Maven:

```bash
mvn test
```

Isso executará todos os testes definidos no projeto.
